# Makgura Website

Production-ready React/Vite landing page for **Makgura**.

## Included

- Full React single-page website
- Premium dark stone / gold game-studio styling
- Faction logos for Rome, Barbarian Horde, and Egypt
- War map pulse / drift effects
- Button shine hover effects
- Responsive layout
- Vercel-ready files

## Local setup

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

Production output goes to `dist/`.

## Deploy to Vercel

1. Upload this folder to a GitHub repo.
2. In Vercel, create a new project and import the repo.
3. Use:
   - Framework: Vite
   - Build command: `npm run build`
   - Output directory: `dist`
4. Deploy.

## Domain

In Vercel: Project → Settings → Domains → Add Domain. Then follow Vercel’s DNS instructions.
